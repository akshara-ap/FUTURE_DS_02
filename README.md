# -FUTURE_DS_02
This Power BI dashboard analyzes telecom customer churn with 7K customers and a 26.58% churn rate. It highlights revenue impact (16.06M total, 2.86M at risk) and identifies high-risk segments by contract type, internet service, tenure, gender, and payment method to support data-driven retention strategies.
📊 Telecom Customer Churn Analysis Dashboard
# 📌 Project Overview

This project presents an interactive Power BI Dashboard that analyzes telecom customer churn data to identify patterns, revenue impact, and high-risk customer segments. The goal is to support data-driven decision-making and improve customer retention strategies.

# 🎯 Objectives

Calculate and monitor overall churn rate
Identify customer segments with high churn probability
Analyze revenue contribution and revenue at risk
Understand churn behavior across contract type, tenure, and services

# 📈 Key Metrics

Total Customers: 7,000
Churned Customers: 2,000
Active Customers: 5,000
Churn Rate: 26.58%
Total Revenue: 16.06M
Revenue at Risk: 2.86M
Average Tenure (Churned): 17.98 months
Average Tenure (Active): 37.65 months

# 📊 Dashboard Insights
# Churn by Contract Type
Month-to-month contracts show the highest churn rate
Two-year contracts have the lowest churn rate

# Churn by Tenure Group
Customers in the 0–12 month range are most likely to churn
Churn rate decreases as tenure increases

# Churn by Internet Service
Fiber optic users have the highest churn rate

# Payment Method Analysis
Electronic check users contribute significantly to churn
Revenue distribution varies by payment type

# Gender Distribution
Churn is almost evenly distributed between male and female customers
